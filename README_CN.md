# MediManage - 医患管理平台

MediManage 是一款专为牙科及医疗门诊打造的高性能患者管理系统。本项目采用模块化架构，支持多语言、多地区动态适配，并拥有极致的视觉设计与流畅的交互体验。

## 🚀 核心功能模块

- **📊 数字化仪表盘**：实时监控核心指标、今日就诊效率及月度患者增长趋势。
- **📅 预约管理系统**：支持日历视图（日/周/月）与看板视图，涵盖预约审核、智能排班与自动提醒。
- **👥 患者档案中心**：全功能患者注册、风险评级（低/中/高）、便捷搜索与全生命周期病历追踪。
- **🏥 诊疗临床路径**：可视化病历管理，支持主诉、检查、诊断方案录入，支持影像图片上传与管理。
- **🦷 专业牙位图 (Dental Chart)**：内置交互式牙位标记器，实现精准的牙齿记录与治疗计划制定。
- **💰 财务结算模块**：追踪月度营收、待支付账单与消费画像分析。
- **🌍 国际化与区域适配**：一键切换中英文环境，针对国内/海外版提供专门的一套 UI 风格与合规字段支持。
- **🔔 全局推送系统**：集成了实时的 Toast 提醒、微信/邮件随访通知配置。

## 🛠️ 技术栈

- **核心框架**：[React 18](https://reactjs.org/)
- **构建工具**：[Vite](https://vitejs.dev/)
- **动画引擎**：[Framer Motion](https://www.framer.com/motion/)
- **图标系统**：[Lucide React](https://lucide.dev/)
- **样式方案**：[Tailwind CSS](https://tailwindcss.com/) & 原生 CSS
- **状态管理**：React Context API

## 📂 项目架构

```text
src/
├── components/     # 可复用UI组件 (侧边栏, 表单, 牙位图等)
├── constants/      # 多语言翻译 (translations.js) 及全局配置
├── contexts/       # 全局状态控制 (RegionContext, ToastContext)
├── styles/         # 全局样式与 Tailwind 样式注入
├── views/          # 页面级视图组件 (仪表盘, 预约, 患者管理等)
└── App.jsx         # 模块化主入口
```

## 🚥 快速开始

### 前提条件

- Node.js (建议使用 LTS 版本)
- pnpm (推荐) 或 npm

### 安装与运行

1. 安装依赖：

   ```bash
   pnpm install
   ```

2. 启动开发服务器：

   ```bash
   pnpm dev
   ```

3. 构建生产环境包：
   ```bash
   pnpm build
   ```

## 🎨 设计原则

- **极致美学**：采用高对比度暗色模式背景、柔和渐变与毛玻璃效果（Glassmorphism）。
- **微交互**：通过细腻的过渡动画与悬停效果提升用户在医疗场景下的操作体验。
- **响应式布局**：流式布局设计，完美适配不同尺寸的 Web 与 PC 屏幕。
